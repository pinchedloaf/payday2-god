payday2-god
-----------
Payday 2 mod to toggle god mode.

Requires
--------
IPHLPAPI.dll, v.3.2.0 2015-07-15:1121 version (included).
- Original Author: sirgoodsmoke

Installation
------------
Just copy to pd2 directory.

F1 - enable god mode.
F2 - disable god mode.

Notes
-----
Payday 2, Update 71 (2015-07-16)
